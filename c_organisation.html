Organisation des textes

Les documents proposés sont de plusieurs formats.

Les documents aux formats source servent à générer (compiler ?) par l'intermédiaire d'une application appropriée les documents aux formats de distribution (on dit aussi format opaque). Le tableau suivant indique les formats et les applications utilisées.

L'unique format de distribution est donc le pdf. Il sert à la visualisation et à l'impression des documents pour tous les types de machines et d'OS. Les documents au format pdf n'ont pas à être modifiés.

Les documents proposés sont de plusieurs types, leurs noms respectent certaines règles.

Les documents primaires sont des sources tex ou mw, ils servent à constituer des documents secondaires pdf (devoirs ou documents "A").

Le tableau suivant présente les conventions de nommage pour les fichiers primaires

Enomfic.tex
 fichier tex constitutif de l'énoncé
Enomfic.mw
 fichier Maple pour former les images à inclure dans l'énoncé : Enomfic_1.ps Enomfic_2.ps, ...
Cnomfic.mw
 fichier tex constitutif du corrigé
Cnomfic.mw
 fichier Maple pour former les images à inclure dans le corrigé : Enomfic_1.ps Enomfic_2.ps, ...

Les documents tex ne sont pas compilables directement, ils doivent être inclus dans un fichier tex source des documents secondaires. Le tableau suivant présente les conventions de nommage pour les documents secondaires.

Anomfic.tex (.pdf)
 input de Enomfic.tex et de (éventuellement) Cnomfic.tex
MaannE.tex (.pdf)
 énoncé du devoir maison année aa numéro nn
MaannC.tex (.pdf)
 corrigé du devoir maison année aa numéro nn
SaannE.tex (.pdf)
 énoncé du devoir surveillé année aa numero nn
SaannC.tex (.pdf)
 corrigé du devoir surveillé année aa numéro nn

L'année scolaire 2004-2005 est désignée par le code 04.
